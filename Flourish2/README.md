## Analytical Visualization: OPA Complaints Dataset

This visualization shows the number of complaints filed each year with the Seattle Office of Police Accountability using unfiltered data from the Seattle Open Data Portal.

Instead of simply showing trends, this chart highlights issues within the dataset itself. The extremely low values in early years (such as 1900–2000) suggest missing or placeholder data rather than actual complaint counts.

This raises questions about the reliability of the dataset over time. The sharp increase in complaints after 2010 may reflect changes in data collection or reporting practices rather than true increases in complaints.

Overall, this visualization demonstrates that while the dataset can show general patterns, it also has limitations that affect how accurately it represents long-term trends.


Source:  
Seattle Open Data Portal – Office of Police Accountability Complaints Dataset  
https://data.seattle.gov/
